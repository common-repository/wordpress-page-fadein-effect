== Description ==

This plugin create a page fadein effect for the whole wordpress site to allow pages to show up smoothly without any "block-in". It makes the large page open more smoothly.


== Installation ==

1. Unzip and Upload `fadein.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
